## Examples

```tsx
// Dramatic blur with slow reveal
<BlurInText
  text="Welcome"
  blurAmount={20}
  duration={1.5}
  stagger={0.1}
/>

// Quick word reveal
<BlurInText
  text="Fast and Clean"
  split="word"
  duration={0.5}
  stagger={0.2}
/>

// Blur out effect
<BlurInText
  text="Goodbye"
  direction="out"
  trigger="hover"
/>
```
