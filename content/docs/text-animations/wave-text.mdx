## Examples

```tsx
// Energetic wave
<WaveText
  text="Party Time"
  amplitude={30}
  frequency={0.6}
  speed={1}
/>

// Calm ocean wave
<WaveText
  text="Peaceful"
  amplitude={10}
  frequency={0.2}
  speed={4}
  direction="up"
/>

// Interactive hover wave
<WaveText
  text="Touch Me"
  trigger="hover"
  amplitude={20}
/>
```
