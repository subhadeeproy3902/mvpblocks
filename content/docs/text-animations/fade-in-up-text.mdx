## Examples

```tsx
// Classic fade up
<FadeInUpText
  text="Welcome to our site"
  direction="up"
  split="word"
/>

// Dramatic letter animation
<FadeInUpText
  text="HELLO"
  split="letter"
  stagger={0.1}
  distance={50}
/>

// Slide from left
<FadeInUpText
  text="Sliding In"
  direction="right"
  duration={0.8}
/>

// Scroll triggered
<FadeInUpText
  text="Scroll to reveal"
  trigger="inView"
  distance={40}
/>
```
