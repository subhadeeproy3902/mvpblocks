## Examples

```tsx
// Premium white shine (default)
<ShinyText
  text="Premium Quality"
  shimmerWidth={200}
  duration={3}
/>

// Gold luxury shine
<ShinyText
  text="Luxury Gold"
  shineColor="rgba(255, 215, 0, 1)"
  shimmerWidth={180}
  duration={2.5}
/>

// Interactive hover shine
<ShinyText
  text="Hover for Shine"
  trigger="hover"
  shimmerWidth={200}
/>

// Extra wide slow shine
<ShinyText
  text="Apple Vision Pro"
  shimmerWidth={250}
  duration={4}
/>

// Diagonal shine
<ShinyText
  text="Angled Shine"
  angle={45}
  shimmerWidth={150}
/>
```
