## Customization

The component uses the `Expandable` UI component which provides:

- **Presets**: `blur-md`, `slide-up` for different animation effects
- **Delays**: Custom expand/collapse delays (200ms expand, 500ms collapse)
- **Sizes**: Collapsed (320x240) and expanded (420x480) sizes
- **Stagger**: Staggered animations for child elements
