## Components Included

### Design Sync Card
A meeting card that expands to show attendees, location, and meeting details with smooth blur animations.

### Product Showcase Card
A product card that expands to reveal detailed features, specifications, and purchase options.

### Weather Forecast Card
A weather card that expands to show detailed weather information and 5-day forecast.

## Customization

The component uses the `Expandable` UI component which provides:

- **Presets**: `fade`, `blur-sm`, `blur-md`, `slide-up`, `slide-down`
- **Custom Animations**: Define your own `animateIn` prop for custom animations
- **Delays**: Control expand/collapse delays for smoother UX
- **Sizes**: Define collapsed and expanded sizes for each card

## Examples

### Custom Expandable Card
```tsx
import {
  Expandable,
  ExpandableCard,
  ExpandableCardHeader,
  ExpandableCardContent,
  ExpandableTrigger,
} from "@/components/ui/expandable";

<Expandable>
  {({ isExpanded }) => (
    <ExpandableTrigger>
      <ExpandableCard
        collapsedSize={{ width: 300, height: 200 }}
        expandedSize={{ width: 500, height: 400 }}
      >
        <ExpandableCardHeader>
          <h3>My Card</h3>
        </ExpandableCardHeader>
        <ExpandableCardContent>
          <p>Expanded content here</p>
        </ExpandableCardContent>
      </ExpandableCard>
    </ExpandableTrigger>
  )}
</Expandable>
```
