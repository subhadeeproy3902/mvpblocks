## Customization

The component uses the `Expandable` UI component which provides:

- **Presets**: `blur-sm`, `slide-up` for different animation effects
- **Delays**: Custom expand/collapse delays (100ms expand, 400ms collapse)
- **Sizes**: Collapsed (300x220) and expanded (500x420) sizes
- **Stagger**: Staggered animations with 0.1s delay between children
- **Spring Animations**: Custom spring animations with rotation effects
