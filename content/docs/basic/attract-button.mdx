## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `particleCount` | `number` | `12` | Number of particles to display |
| `attractRadius` | `number` | `50` | Radius for particle attraction (not currently used in animation) |
| `className` | `string` | - | Additional CSS classes |
| All standard button props | - | - | Extends Button component props |

## Customization

The button uses the project's primary color scheme. You can customize it by:

- **Particle Count**: Adjust `particleCount` prop
- **Colors**: Modify Tailwind classes in the component
- **Animation Speed**: Adjust spring stiffness and damping in the component code
